# Happy Cupcake Premium Demo

A responsive one-page website concept prepared for GitHub Pages or Cloudflare Pages.

## Files
- `index.html`
- `style.css`
- `script.js`
- `assets/happy-cupcake-hero.jpg`
- `assets/happy-cupcake-logo.png`

## Upload to GitHub
1. Create a new repository, for example `happy-cupcake-website`.
2. Upload **the contents of this folder**, not the outer folder itself.
3. Open repository **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select `main` and `/root`, then Save.

## Replace before publishing
Search in `index.html` and replace:
- demo menu names and prices
- Google Maps link
- WhatsApp link and phone number
- opening hours
- verified customer reviews
- exact address and Instagram link

## Notes
The scroll cupcake is a CSS-built pseudo-3D model, so it stays lightweight and works without external animation libraries.
